R Package

To install packages,

  • In RGui: Click on command Menu “\(Packages\)” to select “\(Install\) \(packages\)”.
  • It can be done similarly in RStudio.

To load packages,

  • Cick “\(Packages\)”, and then select “\(Load\) \(packages\)”.
  • Use command “library”
